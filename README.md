# 🚀 GitScope

Analyze GitHub repositories with MCP-powered AI insights.

## 🔍 Features
- Repo health analysis
- PR summaries
- Issue & bug insights
- Structured AI output

## ⚙️ Run locally
export OPENAI_API_KEY="your-key"
export GITHUB_TOKEN="your-token"
python3 -m streamlit run github_agent.py
